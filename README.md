# 17th-team5-Android
디프만 17기 오거덩스팀 안드로이드 프로젝트

### 협업 Rule
- **PR Title**
  - [Feature/Chore/Refactor/Fix] 제목
- **PR Review**
  - 최소 한 명 이상 approve 시, merge 가능
  - pN rule으로 리뷰 남기기
- **Merge and pull rule**
  - squash merge
  - rebase the current branch
- **Jira와 Branch 연결하기**

- minSDK : 29
- compile & targetSdk = 35

- build-logic(CustomConventionPlugins)

- Room

- Coroutine Flow
